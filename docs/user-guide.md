# 🎯 راهنمای کاربری

## 🔹 ورود به سیستم  
۱. مرورگر را باز کنید و به `http://localhost:3000` بروید.  
۲. نام کاربری و رمز عبور خود را وارد کنید.  

## 🔥 ایجاد یک مأموریت جدید  
۱. روی دکمه `+ ایجاد مأموریت` کلیک کنید.  
۲. اطلاعات مأموریت را وارد کرده و ذخیره کنید.
